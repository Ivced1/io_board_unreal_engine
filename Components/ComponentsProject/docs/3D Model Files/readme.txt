1. The IC Chip started out as a solidworks file and was converted to STEP using GrabCad.

2. The STEP file was then converted to GLB using CAD assistant.

3. To get the file to the FBX format, the GLB file was inported into Blender and then exported as FBX to be used in Unreal Engine.


GrabCad - https://grabcad.com/

CAD Assistant - https://www.opencascade.com/products/cad-assistant/ 

Blender - https://www.blender.org/
